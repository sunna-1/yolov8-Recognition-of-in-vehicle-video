# 安装问题解决方案

## 问题：SSL证书错误或代理连接失败

如果遇到以下错误：
- `SSLError` 或 `ProxyError`
- `Could not find a version that satisfies the requirement`

## 解决方案

### 方案1：修复pip配置文件（推荐）

1. 找到pip配置文件位置：
   - Windows: `%APPDATA%\pip\pip.ini` 或 `%USERPROFILE%\pip\pip.ini`
   - 或者在虚拟环境中: `venv\pip.ini`

2. 编辑配置文件，修改或添加以下内容：

```ini
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
trusted-host = pypi.tuna.tsinghua.edu.cn
```

或者使用阿里云镜像：

```ini
[global]
index-url = https://mirrors.aliyun.com/pypi/simple/
trusted-host = mirrors.aliyun.com
```

3. 如果配置了代理但代理不可用，删除或注释掉代理配置：
```ini
# [global]
# proxy = http://proxy.example.com:8080  # 注释掉这行
```

### 方案2：使用conda安装（如果使用conda环境）

```bash
conda install -c conda-forge opencv numpy flask flask-cors pillow pandas
pip install ultralytics
```

### 方案3：逐个安装包

```bash
pip install ultralytics
pip install opencv-python
pip install numpy
pip install flask
pip install flask-cors
pip install pillow
pip install pandas
```

### 方案4：使用国内镜像源（临时）

```bash
# 使用清华镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple --trusted-host pypi.tuna.tsinghua.edu.cn

# 或使用阿里云镜像
pip install -r requirements.txt -i https://mirrors.aliyun.com/pypi/simple/ --trusted-host mirrors.aliyun.com

# 或使用豆瓣镜像
pip install -r requirements.txt -i https://pypi.douban.com/simple/ --trusted-host pypi.douban.com
```

### 方案5：检查网络连接

1. 确保网络连接正常
2. 检查防火墙设置
3. 如果在公司网络，可能需要配置代理
4. 尝试使用手机热点测试

### 方案6：升级pip

```bash
python -m pip install --upgrade pip
```

## 验证安装

安装完成后，验证关键包是否安装成功：

```bash
python -c "import ultralytics; print('YOLO安装成功')"
python -c "import cv2; print('OpenCV安装成功')"
python -c "import flask; print('Flask安装成功')"
```

## 如果仍然无法解决

1. 检查Python版本（需要3.8+）
2. 尝试重新创建虚拟环境
3. 检查是否有杀毒软件阻止网络连接
4. 联系网络管理员检查代理设置
