# 快速启动指南

## 第一步：安装后端依赖

```bash
# 安装Python依赖
pip install -r requirements.txt
```

**注意**：首次运行时会自动下载YOLOv8模型（约6MB），请确保网络连接正常。

## 第二步：启动后端服务

```bash
python run_backend.py
```

看到 "启动后端服务在 http://0.0.0.0:5000" 表示启动成功。

## 第三步：安装前端依赖

打开新的终端窗口：

```bash
cd frontend
npm install
```

## 第四步：启动前端服务

```bash
npm run dev
```

前端将在 `http://localhost:3000` 启动。

## 使用步骤

1. 打开浏览器访问 `http://localhost:3000`
2. 点击"选择视频文件"上传视频，或点击"启动摄像头"进行实时检测
3. 上传视频后点击"开始分析"
4. 等待处理完成，查看分析结果

## 常见问题

### 1. 后端启动失败
- 检查Python版本（需要3.8+）
- 检查是否安装了所有依赖：`pip install -r requirements.txt`
- 检查端口5000是否被占用

### 2. 前端启动失败
- 检查Node.js版本（需要16+）
- 删除 `node_modules` 文件夹，重新运行 `npm install`
- 检查端口3000是否被占用

### 3. 摄像头无法访问
- 确保浏览器允许访问摄像头权限
- 使用Chrome或Edge浏览器
- 检查摄像头是否被其他程序占用

### 4. 视频处理很慢
- 这是正常现象，取决于视频长度和帧数
- 系统默认处理前1000帧，可在 `backend/app.py` 中修改 `max_frames` 参数

## 系统要求

- Python 3.8+
- Node.js 16+
- 现代浏览器（Chrome、Edge、Firefox）
- 至少4GB RAM（推荐8GB+）
- 支持CUDA的GPU（可选，用于加速YOLO检测）
